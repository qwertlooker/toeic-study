## 🚀 快速开始

### 第一步：基线测试
1. 阅读 `01_Baseline/toeic-listening-reading-examinee-handbook.pdf`
2. 完成 `01_Baseline/toeic-listening-reading-sample-test.pdf`
3. 填写 `01_Baseline/baseline_record.csv` 记录薄弱项

### 第二步：安装 Anki
1. 运行 `D:\anki-launcher-25.09-windows.exe` 安装 Anki
2. 导入 `04_Vocabulary_Anki/toeic_core_chunks.csv`
3. 设置每日新词 20-30 条

### 第三步：开始学习
- **听力**：使用 `02_Listening_P3P4/audio/` 进行跟读和精听
- **阅读**：学习 `03_Reading_P5P7/TOEIC_Part5_频出文法30パターン.txt`
- **词汇**：每日复习 Anki 牌组 + `toeic_1000_word_list.txt`
