## 三、执行步骤

### 步骤 1：创建文件夹目录结构

在 `e:\English` 下创建以下目录：

```
e:\English\
├── 01_Baseline\          # 基线模考、答题卡、得分记录
├── 02_Listening_P3P4\    # 按周保存音频、文本、精听笔记
├── 03_Reading_P5P7\      # 分题型保存错题与高频词
├── 04_Vocabulary_Anki\   # 导入词库、固定搭配、易错词
├── 05_Mock_Tests\        # 每 4 周一套模考及复盘
├── 06_Weekly_Review\     # 每周自评与下周调整
├── toeic_20_articles_bilingual.docx
├── toeic_study_execution_pack.docx
└── 托业备考计划.md
```

### 步骤 2：下载 ETS 官方免费材料

- 访问 ETS TOEIC 官网，下载 Examinee Handbook 和 Sample Test
- 保存到 `01_Baseline/` 目录

### 步骤 3：整理核心书目购买清单

- 列出需购买的书目（公式問題集、新公式托业、金のフレーズ）
- 确认购买渠道（京东/淘宝/日亚等）

### 步骤 4：从精背文档提取核心词块

- 使用 Python 脚本解析 `toeic_20_articles_bilingual.docx`
- 提取全部 20 篇的核心词块（English Chunk + 中文释义）
- 输出为 Anki 可导入的 CSV/TSV 格式，保存到 `04_Vocabulary_Anki/`

### 步骤 5：为精背文档生成 TTS 音频

- 使用 TTS 工具（如 edge-tts）为 20 篇英文原文生成音频
- 音频保存到 `02_Listening_P3P4/` 目录

### 步骤 6：下载免费补充语料

- 从 BBC 和 VOA 官网下载有文字稿的听力材料
- 保存到 `02_Listening_P3P4/` 目录

### 步骤 7：配置 Anki

- 安装 Anki（如未安装）
- 导入步骤 4 生成的词块 CSV
- 建立每日复习节奏

### 步骤 8：编写基线记录模板

- 在 `01_Baseline/` 下创建基线记录表格
- 包含 Part 1-7 正确率、主要错因、时间不足情况

## 四、注意事项

1. **优先级排序**：官方材料 > 核心书目 > 工具配置 > 免费语料 > TTS 音频
2. **执行包建议**：先按本包执行 4 周，再根据基线和模考结果微调材料强度
3. **精背文档使用**：优先用于邮件、通知、订单、投诉等职场文体；先精读英文再看中文，每篇先记住 5 个核心词块再做 30 秒口头复述
4. **软件安装**：Anki 默认安装到 D 盘（遵循用户规则）
5. **资料获取**：优先从国内渠道获取，避免网络问题
